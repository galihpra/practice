git add . -> untuk menambahkan perubahan yang terjadi pada folder saat ini
git commit -m "perubahannya untuk apa(dalam rangka apa)" -> simpan perubahan
git status -> menampilkan kondisi folder saat ini
git log --oneline -> menampilkan history pekerjaan pada folder
git push origin main -> untuk mengupload perubahan yang terjadi di komputer local ke github
git pull origin main -> untuk mendownload perubahan dari github ke komputer local
